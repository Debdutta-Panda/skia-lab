# Introduction
Skia is google developed amazing 2d graphics library. The official doc is https://skia.org/docs/, but i feel it is not comprehensive, so i got motivated to have one.

# OS Guidance
As of now this doc is based on windows only.

# Download Skia
You can download skia by following intstructions from https://skia.org/docs/user/download/.

# Build Skia
Skia can be build by following instructions from https://skia.org/docs/user/build/.
